# Módulo 02 — TensorFlow.js Intro

## Pré-requisitos

- Node.js 24.14.0 (LTS) via [nvm-windows](https://github.com/coreybutler/nvm-windows)

## Instalação

```bash
npm install
```

Ou individualmente:

```bash
npm i @tensorflow/tfjs@4.22
```

## Iniciando o projeto

### Modo desenvolvimento (com hot reload)

```bash
npm start
```

## Dependências principais

| Pacote | Versão |
|---|---|
| `@tensorflow/tfjs` | 4.22 |
