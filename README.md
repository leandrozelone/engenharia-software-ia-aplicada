# 🎓 Pós-graduação em Engenharia de Software com IA Aplicada

Este repositório contém materiais, anotações e projetos desenvolvidos durante minha pós-graduação em Engenharia de Software com foco em Inteligência Artificial aplicada.

## 📌 Objetivo

Consolidar conhecimentos em desenvolvimento de software moderno, aliando boas práticas de engenharia com técnicas de Inteligência Artificial para criação de soluções eficientes, escaláveis e orientadas a dados.

## 🚀 Tecnologias e Conceitos

- Arquitetura de Software  
- Clean Code e princípios SOLID  
- Integração com APIs e serviços de IA  
- Machine Learning e análise de dados  
- Automação de processos  
- Testes automatizados  
- Integração e entrega contínua (CI/CD)
